# shoebill_api
 A Grape API mounted on Rails for the graduates in our cohort
 
# Production Host
http://localhost:3000/api/v1/graduates/


# GET /v1/graduates
Returns an array with of all the students

# GET /v1/graduates/1
Returns an array of each student sorted by ID. <br>
